{# Mode-aware note. Shown in track-mid mode to make the "yes_cents is ignored" reality unmistakable. #}
market-following mode: theo = (best_bid + best_ask) / 2 each cycle. Yes (cents) is ignored at quote time. Confidence still picks the strategy mode (penny/match/follow).
Confidence scale
  • ≥ 0.95 — near-certain → penny INSIDE best (bid = best+1, ask = best−1) · most aggressive
  • ≥ 0.70 — trust within ~3¢ → MATCH best (bid = best_bid, ask = best_ask) · sits at LIP reference price
  • ≥ 0.10 — unreliable but quotable → active-follow (1¢ BEHIND best on each side) · safer
  • < 0.10 — strategy skips both sides entirely (knob: min_theo_confidence)
  • 0.00 — stub default, never quotes (set an override to start trading)
Anti-spoofing cap (theo_tolerance_c) still binds at every level: bid won't exceed theo−1+tol, ask won't drop below theo+1−tol.
